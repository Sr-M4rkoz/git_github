## Comandos do Git: 

```git
cd [link_do_projeto](pasta)
git init
git add .
git commit -m "Seu commit..."
git branch -M main
git remote add origin [link_do_repositorio](https://github.com/Sr-M4rkoz)
git push -u origin main
```
